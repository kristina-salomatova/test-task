### 2.2 Исправленная версия ТЗ:
1.	В корзине может находиться не более 5 различных товаров.
2.	Пользователь может добавить в корзину от 1 до 4 единиц одного товара.
3.	Суммарное количество всех товаров в корзине не может превышать 20 штук.
4.	При попытке добавить товар, превышающий лимиты, система показывает сообщение: "Лимит корзины превышен".
5.	Пользователь может изменять количество каждого товара в корзине.
6.	Если пользователь уменьшает количество товара до 0, товар удаляется из корзины.
7.	В корзине может быть реклама других продуктов.
8.	Если цена на товар изменилась в каталоге, система должна автоматически обновить ее в корзине у всех пользователей.
9.	На странице корзины отображается список товаров, их количество, цена за единицу и общая стоимость позиции.
